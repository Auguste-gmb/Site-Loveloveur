# Lovélo Rouen — Site Web

> Projet citoyen · Cycle Ingénieur · Rouen, 2024–2025

Site web de présentation du projet **Lovélo Rouen** : une initiative citoyenne pour encourager les habitants et visiteurs de l'agglomération rouennaise à explorer la ville à vélo, en s'appuyant sur le service de vélos en libre-service [Lovélo](https://lovelolibreservice.fr/).

## Stack & Design

- **HTML / CSS / JavaScript** vanilla — aucune dépendance, aucun framework
- Polices : [Syne](https://fonts.google.com/specimen/Syne) (titres) + [Inter](https://fonts.google.com/specimen/Inter) (corps)
- Couleurs de marque :
    - Magenta `#b00457`
    - Violet `#472977`
- Entièrement responsive — mobile, tablette, desktop

## Déploiement sur GitHub Pages

1. **Forker ou cloner** ce dépôt
2. Aller dans **Settings → Pages**
3. Source : `Deploy from a branch` → branche `main` → dossier `/ (root)`
4. Cliquer **Save** — le site est en ligne en quelques minutes

> Les 4 fichiers (`index.html`, `style.css`, `script.js`, `logo.png`) doivent être à la **racine** du dépôt.

## Application mobile

L'application Lovélo Rouen est actuellement **en cours de développement**. Elle permettra de :

- Suivre les tracés GPS en temps réel
- Rejoindre des sorties groupées
- Localiser les stations Lovélo à proximité
- Découvrir les points d'intérêt sur les parcours
